# Next App

Boilerplate for React Application

## Usage
+ `git clone`
+ `rm -rf .git`
+ `git init`
+ `yarn`
+ `yarn dev`

### Start dev
+ `yarn build`

### Checks
+ `yarn test` to run unit tests
+ `yarn lint:code` to check code-style
+ `yarn lint:style` to check style-style

### Commit
+ `yarn s cz` to commit

### Storybook
+ `yarn storybook` to run storybook (on http://localhost:9001)
